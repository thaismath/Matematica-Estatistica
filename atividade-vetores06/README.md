# Álgebra Linear: Calculando a Similaridade Entre Vetores de Dados

Este repositório contém a resolução de exercícios práticos focados em **Vetores**, parte do módulo de Matemática e Estatística Aplicada do curso de Formação em Ciência de Dados (Data Science Academy).

A proposta desta atividade é unir os fundamentos teóricos de Álgebra Linear com a aplicação prática dos estudos do capítulo 6 - Vetores e Espaço Vetorial. 

##  Objetivos do Exercício
- Implementar operações fundamentais de Álgebra Linear.
- Explorar o uso da biblioteca `NumPy`, `Math`, `Sympy` para otimização de cálculos vetoriais e `matplotlib` para visualização de vetores.
- Traduzir conceitos matemáticos para código.
- Interpretação para o uso em análises de dados. 

##  Tópicos Abordados
- **Estruturas de Dados:** Definição de vetores usando listas nativas e arrays (`numpy.array`).
- **Operações Aritméticas:** Adição, subtração e multiplicação por escalar.
- **Produto Escalar (Dot Product):** Cálculo da projeção e do alinhamento entre vetores.
- **Norma do Vetor:** Cálculo da magnitude do vetor no espaço.
- **Otimização:** Encontrar as projeções de maior variância e classificar dados.
- **Similaridade:** Quantificar o grau de semelhança entre dois ou mais vetores ou conjuntos de dados.
